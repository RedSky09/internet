Memahami Cara Kerja Internet

Internet

Internet adalah jaringan komputer global yang menghubungkan miliaran perangkat di seluruh dunia. Berikut adalah cara kerjanya secara umum:

Protokol TCP/IP

Internet bekerja menggunakan protokol TCP/IP (Transmission Control Protocol/Internet Protocol)

Paket Data

Data dipecah menjadi paket-paket kecil yang dikirim melalui berbagai rute, kemudian paket data disusun kembali di tujuan menjadi informasi yang utuh

Routing

Router dan switch mengarahkan paket data ke tujuan yang tepat

Sumber: Internet Society (ISOC), "How Internet Works", 2023

HTTP vs HTTPS

HTTP

  • Protokol dasar untuk transfer data di web
  • Bekerja dengan model request-response
  • Data dikirim dalam bentuk plaintext

Sumber: Mozilla Developer Network (MDN), "HTTP vs HTTPS", 2024

HTTPS

  • Versi aman dari HTTP dengan enkripsi SSL/TLS
  • Melindungi data sensitif selama transfer
  • Menggunakan sertifikat digital untuk autentikasi

Nama Domain

Apa itu domain

Domain adalah alamat yang digunakan untuk mengakses website. Struktur domain terdiri dari:

Top-level Domain

.com
.org
.net

Second-level Domain

Nama website

example.com

Subdomain

Bagian tambahan sebelum nama domain

blog.example.com

Sumber: ICANN, "Domain Name System Explained", 2024

Hosting

Hosting adalah sebuah layanan online yang digunakan untuk mengelola data situs atau aplikasi web oleh pengguna dan ditampilkan melalui via internet. File website yang tersimpan di dalam hosting berupa gambar, script, video, email, aplikasi, dan database (basis data)

Shared Hosting

berbagi server dengan website lain

VPS (Virtual Private Server)

server virtual dedikasi

Dedicated Hosting

server fisik khusus

Cloud Hosting

hosting berbasis cloud computing

Sumber: https://www.sekawanmedia.co.id/blog/pengertian-hosting/

DNS (Domain Name System)

DNS adalah sistem yang menerjemahkan nama domain menjadi alamat IP. Proses DNS meliputi:

1. Browser meminta resolusi DNS

2. DNS Resolver mencari di cache lokal

3. Jika tidak ditemukan, mencari di Root DNS Server

4. Dilanjutkan ke TLD DNS Server

5. Akhirnya ke Authoritative DNS Server

Sumber: Cloudflare, "What is DNS?", 2024

Browser

Browser adalah software yang digunakan untuk mengakses web. Cara kerja browser:

1. Parsing HTML, CSS, dan JavaScript

2. Membuat DOM (Document Object Model)

3. Rendering halaman web

4. Menjalankan script

5. Mengelola cache dan cookies

Sumber: Google Chrome Developers, "Inside Browser", 2024
